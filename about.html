This is a POC standalone app which demonstrate how to manage Competency Frameworks from Moodle as .csv files.
At first, it was supposed to be a simple previewer para los gringos and amigos, which would allow to upload a .csv file and display it as you would see it in Moodle if you were god (or an admin).
But then, it quickly evolved into a full-fledged app that could races to the top 1 of Product Hunts, if anyone cared about education stuff as more than just an economical valuable value.
Now, you can use multiple inputs (including original .csv files from moodle, direct distant .csv urls, JSON inputs, and a competency framework Generator...), multi-selections, multi-edits and exports etc.
In the end, this is just an attempt to remember us what does modular object-oriented dynamic learning environment actually means.
